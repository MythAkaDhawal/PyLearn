🌐 Introduction

Python's Superpower: Its Ecosystem

What makes Python the world's most popular programming language isn't just the language itself — it's the massive ecosystem of free libraries built by developers around the world. There are over 450,000 free packages available. Whatever you want to build, someone has probably already built the hardest parts for you.

Python is popular because it doesn't make you build everything from scratch. Data science, web scraping, AI, building websites — there's a library for all of it! 🚀
📦 pip — Python's App Store

Installing libraries with pip

pipPython's package manager. Use it in your terminal to download and install libraries from the internet. is Python's package manager — think of it as an App Store for Python libraries. You run it in your terminal (not inside Python code) to install new libraries. 

# Run this in your TERMINAL (not Python!):
pip install numpy
pip install pandas
pip install requests

# Then import it inside your Python code as usual:
import numpy as np
💡 Pro Tip: The as np part is an alias — just a shorter nickname so you don't have to type numpy every time. import numpy as np and import pandas as pd are industry-wide conventions.
🔢 NumPy — Number Crunching

Fast math with huge datasets

NumPy (Numerical Python) is the foundation of almost all data science in Python. It introduces the array — a super-powered list that can do math on millions of numbers simultaneously, hundreds of times faster than a regular Python loop.

numpy_demo.py 
import numpy as np

scores = np.array([88, 92, 79, 95, 85])

print("Average:", np.mean(scores))
print("Max:", np.max(scores))
print("Min:", np.min(scores))
print("Doubled:", scores * 2)
3np.array([...])Creates a NumPy array. It looks like a list, but it has math superpowers.
8scores * 2Doubles every single number in the array at once. Try doing that easily with a regular Python list!
🐼 Pandas — Working with Tables

Python's Excel

Pandas lets you work with tabular data (like a spreadsheet) directly in Python. It introduces the DataFrame — a table with rows and columns that you can filter, sort, group, and analyse with incredibly simple code.

pandas_demo.py 
import pandas as pd

data = {
    "Name":  ["Alice", "Bob", "Charlie"],
    "Score": [88, 92, 79]
}

df = pd.DataFrame(data)
print(df)
print("\nAverage Score:", df["Score"].mean())
8pd.DataFrame(data)Turns the dictionary into a proper table with rows and columns.
10df["Score"].mean()Select the "Score" column and instantly calculate the average — no loop needed!
🌍 Requests — Talking to the Internet

Fetching data from websites and APIs

The requests library lets your Python program act like a web browser — it can fetch web pages and talk to APIs (Application Programming Interfaces) to retrieve real-time data like weather, cryptocurrency prices, or news headlines. 

import requests

# Fetch data from a public API
response = requests.get("https://api.github.com")

print("Status:", response.status_code)  # 200 = OK!
print("Response:", response.json())      # Get data as a dict
⚠️ Browser Note: The requests library requires an actual internet connection and runs locally. You can't run it in our in-browser playground, but it works perfectly in any local Python install!
